# 🛠️ סקילים לקלוד — מתנה מטירונות סוכנים

> **18 סקילים מוכנים לשימוש** שיהפכו את Claude לכלי חזק יותר.
> כל סקיל הוא יכולת חדשה שמלמדת את Claude לעשות משהו ספציפי — חשיבה עמוקה, ניתוח אנשים, בניית אסטרטגיה, ועוד.

---

## 📦 איך מתקינים

### שלב 1 — הורידו את הקובץ
לחצו על הקישור שקיבלתם במייל להורדת ה-ZIP.

### שלב 2 — חלצו את התיקייה
פתחו את ה-ZIP. בפנים תמצאו תיקיות — כל תיקייה היא סקיל אחד.

### שלב 3 — התקינו
העתיקו את התיקיות שרוצים לתיקייה:
```
~/.claude/skills/
```

**ב-Mac:** פתחו Terminal והריצו:
```bash
cp -r שם-הסקיל ~/.claude/skills/
```

**לא בטוחים איפה זה?** הנתיב המלא הוא:
```
/Users/שם-המשתמש-שלכם/.claude/skills/
```

### שלב 4 — הפעילו
פתחו שיחה חדשה ב-Claude Desktop. הסקילים נטענים אוטומטית.
כל סקיל מופעל כשאתם אומרים את מילות ההפעלה שלו (מפורטות למטה).
