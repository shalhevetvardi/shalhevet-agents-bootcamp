# יצירת מיילים HTML מעוצבים עם AI

## מה זה עושה
מגדיר prompt שמלמד AI ליצור מיילים מעוצבים ב-HTML ישירות ב-Gmail — עם צבעים, באנרים, בלוקים צבעוניים ואמוג'י, בלי כלי עיצוב חיצוני.

## למה זה שימושי
מיילים שנראים טוב מקבלים יותר תשומת לב. במקום לעצב ב-Canva או ב-Mailchimp — ה-AI יוצר HTML מעוצב שנכנס ישירות ל-Gmail. מושלם להצגת תוצאות ללקוח, הזמנות, עדכוני סטטוס.

## מה צריך
- [ ] Claude / ChatGPT / AI אחר שיודע לייצר HTML
- [ ] Gmail (או כל לקוח מייל שתומך ב-HTML)
- [ ] אופציונלי: Gmail MCP/API (לשליחה אוטומטית)

## איך משתמשים

### שימוש ידני
1. העתיקו את ה-prompt מ-`styled-email-prompt.md` ל-AI
2. בקשו ממנו לכתוב מייל על הנושא שלכם
3. העתיקו את ה-HTML שנוצר לחלון compose ב-Gmail

### שימוש עם AI Agent
אם יש לכם סוכן AI שמחובר ל-Gmail (MCP / API):
- הסוכן קורא את ה-prompt
- מזהה את סוג המייל (תוצאות / סטטוס / הזמנה / ידע / אישי)
- בוחר פלטת צבעים מתאימה
- מייצר HTML ושולח ישירות

## איך להתאים אישית

### הוספת פלטה חדשה
הוסיפו שורה לטבלת הפלטות ב-`styled-email-prompt.md`:
```
| **שם** | `#color1 → #color2` | מתי משתמשים |
```

### שינוי סוגי מיילים
הוסיפו / שנו שורות בטבלת סוגי המייל. לכל סוג הגדירו "דפוס עיצוב" — מה המבנה הוויזואלי.

### שפה אחרת
שנו `dir="rtl"` ל-`dir="ltr"` ב-template אם כותבים באנגלית.

## מבנה הקבצים
```
styled-email-draft/
├── styled-email-prompt.md     ← ה-prompt לAI
├── examples/
│   ├── project-results.html   ← דוגמה: הצגת תוצאות (סגול)
│   ├── event-invitation.html  ← דוגמה: הזמנה (כתום-ורוד)
│   └── status-update.html     ← דוגמה: עדכון סטטוס (ירוק)
└── README.md
```

## דוגמאות
פתחו את קבצי ה-HTML בתיקיית `examples/` בדפדפן כדי לראות איך הם נראים. כל דוגמה כוללה תגובה בראש הקובץ שמסבירה באיזו פלטה ואיזה סוג מייל היא משתמשת.
