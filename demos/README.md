# דוגמאות אוטומציות AI — שלהבת ורדי

> אוסף של כלים ואוטומציות שבניתי לעסק שלי עם AI.
> כל דוגמה כאן מנוקה ומוכנה להתאמה — קחו, שנו, והשתמשו.

---

## מה יש כאן

| תיקייה | מה זה | סוג |
|---------|--------|-----|
| `zoom-workflow/` | סיכום פגישות זום אוטומטי | Bash + Python |
| `b-roll-generator/` | יצירת תמונות וסרטונים מתסריט | Python + Gemini API |
| `content-analytics/` | ניתוח ביצועי תוכן שיטתי | Skill (מתודולוגיה) |
| `email-auto-responder/` | מענה אוטומטי למיילים | Python + Claude API |
| `video-editor/` | הסרת שקטים + חיתוך לפי נושאים | Python + ffmpeg |
| `styled-email-draft/` | יצירת מיילים HTML מעוצבים | Skill (prompt) |

---

## איך להשתמש

1. **בחרו דוגמה** שמעניינת אתכם
2. **קראו את ה-README** שלה — שם מוסבר מה צריך להתקין
3. **העתיקו את התיקייה** לפרויקט שלכם
4. **התאימו** — החליפו את ה-placeholders בפרטים שלכם
5. **הריצו** — בדרך כלל `python3 script.py` או שיחה עם Claude

---

## דרישות כלליות

- Python 3.10+
- מפתח API של Claude (Anthropic) ו/או Gemini (Google)
- כל דוגמה מפרטת דרישות נוספות ב-README שלה
