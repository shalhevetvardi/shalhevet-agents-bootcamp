## שלב 1 — ניתוח כמותי

### 1.1 טבלת דירוג רב-ממדית
כל פריט ידורג ב-4 ממדים:

| ממד | מדד |
|------|------|
| חשיפה | Views |
| שימור | שיעור שימור (Avg Watch / Length) |
| ויראליות | מקדם ויראליות (Shares / Views) |
| אפקטיביות | אפקטיביות CTA (Clicks / Views) |

**שאלה מרכזית:** האם אותם פריטים מצליחים בכל הממדים, או שיש הפרדה?

### 1.2 מתאמים (רק Organic)
- Views <-> Shares — מנוע צמיחה
- שיעור שימור <-> Views — האם שימור = חשיפה?
- אורך <-> שיעור שימור — אורך אופטימלי
- Comments <-> Shares — שיח מול העברה

### 1.3 Paid vs Organic
- הבדלי ביצועים — ממוצעים לכל קבוצה
- האם Paid דוחף צפיות אבל לא שיתופים?

### תוצר
Top 5 / Bottom 5 + 5-8 תובנות כמותניות

## שלב 2 — ניתוח Hooks (פתיחות)

### 2.1 סיווג מבני
קטלגו כל Hook לקטגוריה:

| סוג Hook | דוגמת פטרן |
|----------|-------------|
| שאלה ישירה | "מכירים את זה כש...?" |
| הבטחה / claim | "הנה הטריק ש..." |
| פרובוקציה | "רוב האנשים לא יודעים..." |
| סיפור / רגע | "קרה לי משהו..." |
| הדגמה מיידית | תוצאה מהשנייה הראשונה |
| קונטרה-אינטואיטיבי | "תפסיקו לעשות X" |
| מספר / רשימה | "5 דרכים ש..." |

### 2.2 ניתוח עדין
- אורך Hook (מילים) <-> ביצועים
- כאב ספציפי מול כללי — מה עובד?
- Hook <-> שימור — צפיות גבוהות + שימור נמוך = clickbait

### תוצר
מפת Hooks — מה עובד, למה, אילו פטרנים לחזק

## שלב 4 — ניתוח תמלולים (עומק)

### סדר עבודה

**4א — קיצוניים (Top 3 + Bottom 3)**
1. קראו תמלולים של 3 הפריטים הכי מצליחים + 3 הכי חלשים
2. נתחו: נושא, צורך, מבנה, טון, CTA, twist
3. חפשו contrast — מה שונה?
4. גבשו 5-10 השערות

**4ב — אימות רוחבי**
1. סרקו שאר התמלולים לאמת/להפריך
2. מפו כל פריט ל: מבנה, נושא, טון, CTA
3. ציינו "חוזק ההשערה"

### עדשת אסטרטגיה
- מה הנושא האמיתי
- איזה צורך של הקהל נענה
- מה הרגע שהקהל מזדהה
- מה הזווית שגרמה לתגובה

### עדשת כתיבה
- פייסינג — מהירות, קצב
- טון — שיחתי? מקצועי? פגיע?
- מבנה פנימי — פתיחה, גוף, סיום
- CTA — איפה, איך, מה
- Twist / נקודת מפנה
- אורך אפקטיבי — איפה נושרים

## רמות ביטחון
- **חזק** — חוזר ב-3+ פריטים
- **בינוני** — חוזר ב-2
- **השערה** — פעם אחת, צריך אימות

## הנחיות חשובות
- תמיד הפרידו Paid מ-Organic
- גישת "דלתא" — לא מחליפים, רק מוסיפים/מעדכנים
- ויראליות ≠ אפקטיביות — המטרה מכתיבה את ההערכה
