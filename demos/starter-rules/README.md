# Starter Rules — כללים להתחלה

> כללים בסיסיים שכל תלמיד מכניס ל-CLAUDE.md שלו בשיעור 1.
> אלה נקודת התחלה — לא הסוף. כללים גדלים עם הזמן.

---

## מה זה כלל

כלל = הוראה שקלוד מקבל **בתחילת כל שיחה**.
במקום להגיד כל פעם מחדש "דבר איתי בעברית" — כותבים את זה פעם אחת ב-CLAUDE.md.

## הכללים שמתחילים איתם

### כלל 1: שפה
```
דבר איתי בעברית. כל הפלטים בעברית.
```

### כלל 2: שאל לפני שמתחיל
```
לפני שמתחיל משימה — וודא שהבנת מה אני רוצה. "האם זה הכיוון?" ואז המשך.
```

### כלל 3: שמור פלט ביניים
```
כל תוצר משמעותי — שמור לקובץ, לא רק לצ'אט. תוצרים לא נשארים בשיחה שתיסגר.
```

### כלל 4: אל תמציא
```
אם אין לך מידע — אמור שאין. אל תמציא. אם צריך מידע — בקש ממני.
```

### כלל 5: אישור לפני פעולות יקרות
```
מחיקה, שליחה, push, פרסום — תמיד אישור ממני לפני.
```

---

## איך מכניסים

1. פותחים את `CLAUDE.md` בשורש הריפו
2. מוסיפים את הכללים תחת סקשן "כללי עבודה"
3. שומרים — מהרגע הזה, קלוד מקבל אותם בכל שיחה

## מה יקרה אחרי

עם הזמן תוסיפו עוד כללים — העדפות אישיות, סגנון כתיבה, מה אסור.
הכללים גדלים. זה נורמלי.
