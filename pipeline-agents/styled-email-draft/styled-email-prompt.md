# Styled Email Draft — Prompt Template

## מתי משתמשים

- מייל "מעוצב", "יפה", "מרשים", "עם עיצוב"
- מייל שצריך להרשים לקוח / שותף / קהל
- כשההקשר מצריך תצוגה ויזואלית (תוצאות פרויקט, דוח, הצגת עבודה)

## הוראות

### שלב 1 — זיהוי סוג המייל

קבע את סוג המייל לפי הטבלה:

| סוג | מתי | דפוס עיצוב |
|-----|-----|-------------|
| **הצגת תוצאות** | מציגים עבודה / פרויקט שהושלם | באנר + רשימת שלבים צבעונית + דוגמאות |
| **עדכון סטטוס** | דיווח על התקדמות | טבלת סטטוס + אייקונים ירוק/צהוב/אדום |
| **הזמנה / הצעה** | הזמנה לאירוע, הצעה עסקית | באנר בולט + CTA ברור + תאריכים |
| **שיתוף ידע** | שליחת מדריך, הסבר, תוכן מקצועי | כותרות מחולקות + בלוקים צבעוניים + טיפים |
| **מייל אישי מרשים** | תודה, ברכה, הודעה אישית | עיצוב עדין, צבעים רכים, אמוג'י מדוד |

### שלב 3 — בנייה

מבנה HTML בסיסי:

```html
<div dir="rtl" style="font-family: Arial, sans-serif; font-size: 15px; line-height: 1.8; color: #222;">

  <!-- באנר עליון (אופציונלי — להדגשת מסר מרכזי) -->
  <div style="background: linear-gradient(135deg, COLOR1, COLOR2); color: white; padding: 20px 24px; border-radius: 12px; margin-bottom: 24px; text-align: center;">
    <p style="font-size: 13px; margin: 0;">MAIN_MESSAGE</p>
  </div>

  <!-- תוכן -->

  <!-- קו מפריד -->
  <hr style="border: none; border-top: 1px solid #e0e0e0; margin: 20px 0;">

  <!-- כותרת משנית -->
  <h3 style="color: PRIMARY_COLOR;">EMOJI TITLE</h3>

  <!-- בלוק צבעוני (לפריטים ברשימה) -->
  <div style="padding: 10px 14px; background: LIGHT_BG; border-radius: 8px; margin-bottom: 6px;">
    EMOJI <strong>LABEL</strong> — DESCRIPTION
  </div>

</div>
```

### כללי עיצוב

| כלל | הסבר |
|-----|------|
| **`dir="rtl"`** | חובה — המיילים בעברית |
| **`font-family: Arial, sans-serif`** | בסיס — נתמך בכל הלקוחות |
| **Inline styles בלבד** | Gmail מתעלם מ-`<style>` tags |
| **`border-radius: 8-12px`** | נותן תחושה מודרנית |
| **`line-height: 1.8`** | קריאות בעברית |
| **רקעים בהירים לבלוקים** | `#f3f0ff`, `#f0fdf4`, `#fff7ed`, `#fdf2f8`, `#eff6ff` |
| **אמוג'י בכותרות** | כותרת אחת = אמוג'י אחד בהתחלה |
| **לא יותר מ-3 צבעי רקע שונים** | אחרת נראה עמוס |

### שלב 4 — בדיקת שלמות

- [ ] קישורים ב-`<a href>` ולא טקסט חשוף
