# עריכת וידאו אוטומטית

## מה זה עושה
מסיר שקטים ארוכים מסרטונים, ואופציונלית חותך רק נושאים נבחרים. בלי תוכנת עריכה, בלי ידע טכני.

## למה זה שימושי
הקלטתם שיעור / פגישה / הדרכה — ויצא סרטון של 40 דקות עם 10 דקות של "אממ..." ושקטים. הסקריפט מוריד את כל ה-dead air אוטומטית, ובמצב מתקדם — גם חותך רק את הנושאים הרלוונטיים.

## מה צריך
- [ ] Python 3.8+
- [ ] ffmpeg + ffprobe מותקנים (`brew install ffmpeg` ב-Mac)
- [ ] קובץ וידאו (mp4, mov, mkv)
- [ ] אופציונלי: קובץ topics.json (לחיתוך לפי נושאים)

## איך מריצים

### מצב 1: סרטון מלא נקי (רק הסרת שקטים)
```bash
python3 scripts/edit_video.py video.mp4 --silence-only
```

### מצב 2: סרטון מצומצם (בחירת נושאים + הסרת שקטים)
```bash
python3 scripts/edit_video.py video.mp4 --topics examples/topics.json
```

### מצב אינטראקטיבי
```bash
python3 scripts/edit_video.py video.mp4
```
הסקריפט ישאל מה רוצים ויציג את הנושאים לבחירה.

## הפלט
קובץ חדש באותה תיקייה: `video - ערוך.mp4`

## איך להתאים אישית

### כוונון זיהוי שקתים
בראש `scripts/edit_video.py`:
```python
SILENCE_NOISE_DB = -28       # סף רעש (dB). נמוך יותר = רק שקט מוחלט
SILENCE_MIN_DURATION = 0.8   # מינימום שניות שקט לחיתוך
PADDING = 0.15               # ריפוד לפני/אחרי דיבור
```

### קובץ נושאים
צרו JSON בפורמט שמופיע ב-`examples/topics.json`. כל נושא צריך `name`, `start`, `end`. תתי-נושאים (subtopics) אופציונליים.

### שילוב עם תמלול
אם יש לכם Whisper API — תמללו את הסרטון, תנו ל-AI לזהות נושאים ולייצר topics.json, ואז תריצו את הסקריפט. כך מקבלים עריכה אוטומטית מלאה.

## מבנה הקבצים

```
video-editor/

```

## הזרימה
```
[1] בחירת מצב (מלא נקי / מצומצם)
       ↓
[2] אם מצומצם: טעינת topics.json → בחירת נושאים → חיתוך
       ↓
[3] זיהוי שקטים (ffmpeg silencedetect)
       ↓
[4] חיתוך קטעי דיבור + חיבור
       ↓
[5] קובץ ערוך מוכן 🎉
```
