## 📋 11 הפאנלים

### **PANEL 1** — האינסטגרם בבוקר

- **טקסט וויסאובר:** *"את פותחת את האינסטגרם בבוקר..."*
- **Scene description:** Close-up מלמעלה של יד נשית מחזיקה אייפון. מסך מציג סטורי "Claude Code 🤯" בגרדיאנט סגול-ורוד. רקע מטושטש: שולחן עץ, ספל קפה אדים, מחברת.
- **Camera angle:** Top-down 45° (over-the-shoulder)
- **Lighting:** אור בוקר רך משמאל, חלון מסונן
- **Emotion:** Curious, mildly distracted
- **Duration:** 2-3s

---

### **PANEL 2** — סטורי ראשון Claude Code

- **טקסט וויסאובר:** *"בסטורי הראשון — מישהי מספרת על Claude Code."*
- **Scene description:** Zoom-in למסך הטלפון. סטורי במלואו: אישה ב-30+ עם הכיתוב "I built my first agent with Claude Code 🚀".
- **Camera angle:** Phone screen close-up
- **Lighting:** אור מסך טלפון בלבד (cyan/blue glow)
- **Emotion:** Mild comparison
- **Duration:** 2s

---

### **PANEL 3** — גוללת ללינקדאין

- **Scene description:** האישה עוברת מאינסטגרם ללינקדאין. פוסט: גבר בחליפה, "How Claude Code transformed my workflow", 247 לייקים.
- **Camera angle:** POV (over the shoulder)
- **Lighting:** מטבח, תאורת תקרה בהירה. עומדת.
- **Emotion:** Beginning of unease
- **Duration:** 2-3s

### **PANEL 4** — וואטסאפ קישור לקורס

- **Duration:** 2s

### **PANEL 5** — "כולם כבר שם" (הרגע הפנימי)

- **טקסט וויסאובר:** *"את חושבת לעצמך: 'כולם כבר שם. אני נשארת מאחור.'"*
- **Scene description:** Pull-back רחב. האישה במטבח, אוחזת טלפון רפוי, מבט אבוד. ספל קפה נשכח על השיש. מונסטרה ירוקה ברקע.
- **Camera angle:** Wide medium shot, eye level
- **Lighting:** אור חלון רך מאחוריה — rim light עדין על הכתף והשיער. פנים בצללים קלים.
- **Emotion:** Quiet anxiety / inadequacy
- **Duration:** 3-4s
