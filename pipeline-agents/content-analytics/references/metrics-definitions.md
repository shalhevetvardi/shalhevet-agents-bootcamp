## מטריקות נגזרות — לחשב לכל פריט

| # | מטריקה | נוסחה | מה זה מודד |
|---|---------|-------|------------|
| 1 | **שיעור שימור** | `Avg Watch Time / Video Length` | כמה מהתוכן באמת צפו |
| 2 | **מקדם ויראליות** | `Shares / Views` | "חבר שלי חייב לראות" |
| 3 | **עומק מעורבות** | `Comments / Views` | צורך להגיב |
| 4 | **יחס רגשי/ערכי** | `Likes / Shares` | גבוה=רגשי, נמוך=ערכי |
| 5 | **אפקטיביות CTA** | `Total Clicks / Views` | כמה עשו פעולה |
| 6 | **Reach-to-View** | `Views / Reach` | יעילות הפיד |

## טיפול בבעיות נפוצות

### המרת Views (אם בטקסט)
```
"1,234" → 1234
"1.2K" → 1200
"1.2M" → 1200000
"" או null → סמן כחסר
```

### המרת אורך תוכן
```
"0:45" → 45
"1:30" → 90
"45 שניות" → 45
```

### חלוקה באפס
אם Views = 0, לא לחשב מטריקות — סמן N/A.

### סיווג Paid/Organic
```
אם Ad CPM > 0 או Ad Impressions > 0 → "Paid"
אחרת → "Organic"
```
